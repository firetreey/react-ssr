# react-ssr
> a simple react-ssr demo
## Commands
- ### Project setup
```
npm install
```
- ### Start and hot-reloads by one command
```
npm run start
```
- ### Pack and hot-reloads for server
```
npm run dev:server
```
- ### Pack and hot-reloads for client
```
npm run dev:client
```
- ### Compiles and hot-reloads the project
```
npm run start
```
